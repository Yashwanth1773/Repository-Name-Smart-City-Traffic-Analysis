# Smart City Traffic Analysis Dashboard

## Project Description

This project analyzes traffic data using Microsoft Power BI.

## Tools Used

* Microsoft Power BI
* Excel

## Features

* Traffic trend analysis
* City-wise traffic comparison
* Accident hotspot identification
* Weather impact analysis

## Dataset

Traffic dataset with 1000 rows containing:
Date, Time, Location, Vehicle Count, Speed, Accidents, Weather.

## Dashboard

The dashboard helps identify traffic congestion and accident-prone areas.
